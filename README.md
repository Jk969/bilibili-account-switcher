# B站账号切换助手

这是一个用于 Edge/Chrome 浏览器的插件，可以帮助你在多个 Bilibili 账号之间一键切换。

## 功能特点

- **一键切换**：保存多个账号，点击即可切换。
- **自动获取**：自动获取当前登录账号的 Cookie 和用户信息（头像、昵称）。
- **安全存储**：账号信息仅存储在本地浏览器中 (`chrome.storage.local`)，不上传任何服务器。
- **简单易用**：简洁的 Popup 界面。

## 安装方法

1. 打开 Edge 浏览器，进入扩展管理页面 (`edge://extensions/`)。
   或者 Chrome 浏览器 (`chrome://extensions/`)。
2. 打开页面左下角或右上角的 **"开发人员模式"** 开关。
3. 点击 **"加载解压缩的扩展"** (Load unpacked)。
4. 选择本项目的文件夹 `bilibili-account-switcher`。
5. 安装完成后，在浏览器工具栏找到 B 站图标即可使用。

## 使用说明

### 1. 添加第一个账号
1. 在 B 站网页上登录你的账号 A。
2. 点击插件图标打开面板。
3. 点击 **"添加当前账号"** 按钮。
4. 账号 A 会出现在列表中。

### 2. 添加更多账号（重要！）
为了保证旧账号 Cookie 不失效，**请勿点击 B 站网页右上角的“退出登录”**。请按以下步骤操作：

1. 点击插件面板上的 **"登录新账号"** 按钮。
   * 这一步会清除浏览器本地的登录状态，但不会通知 B 站服务器注销旧 Cookie。
2. 浏览器页面会自动刷新，变为未登录状态。
3. 在网页上正常登录账号 B（扫码或密码）。
4. 登录成功后，再次点击插件的 **"添加当前账号"**。
5. 现在列表中就有两个有效的账号了。

### 3. 切换账号
- 点击列表中的账号 A，网页会自动刷新并切换回账号 A。
- 点击账号 B，切换回账号 B。

### 4. 删除账号
- 点击账号卡片右侧的 **"×"** 按钮即可删除该账号记录。

## 常见问题

**Q: 为什么切换回去后显示未登录？**
A: B 站的 Cookie 有有效期，如果长时间未登录可能会过期。另外，如果你在网页上点击了“退出登录”，服务器会注销该 Cookie，导致插件中保存的记录失效。请务必使用插件的“登录新账号”功能来切换。

**Q: 账号信息安全吗？**
A: 本插件代码开源，所有数据仅存储在你的本地浏览器中，不会上传到任何第三方服务器。

## 注意事项

- 请勿在不可信的公共电脑上使用。
